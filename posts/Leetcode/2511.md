# Solution

  找相临的两个 -1和1 它们之间0最多的就是答案。

  ```python
  from typing import List

  class Solution:
      def captureForts(self, forts: List[int]) -> int:
          ans = 0
          pre = -1
          for i in range(0, len(forts)):
              if (forts[i] == 1 or forts[i] == -1):
                  if (pre >= 0 and forts[i] != forts[pre]):
                      ans = max(ans, i - pre - 1)
                  pre = i
          return ans
  ```
